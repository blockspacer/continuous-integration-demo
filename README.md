# travisci-demo
A demo project using TravisCI with C++ projects

## Manual Usage 

I case you want to test Google Test is working correctly

```bash
git clone --recursive https://github.com/daixtrose/travisci-demo
cd travisci-demo/
mkdir build
cd build
cmake ..
```
